# LandingPage

Technology stack:

- HTML
- SCSS
- Webpack
- Javascript es6+

---

### Command to run project:

1. clone 'repo link'
2. checkout to 'develop' branch
3. npm install
4. npm run prod(for production mode)
5. npm run dev(for development mode)
